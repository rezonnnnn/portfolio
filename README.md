# Zikri Azri Zahairi - Portfolio

Professional portfolio website showcasing data analytics projects, technical skills, and career experience.

**Live Site:** https://rezonnnnn.github.io/portfolio

## About

Analytics Engineer specializing in data pipelines, business intelligence, and automation. Experienced with Databricks, PySpark, Power BI, and SQL.

## Built With

- React 18 + TypeScript
- SCSS for styling
- Material-UI components
- Deployed on GitHub Pages

## Features

- Responsive design optimized for all devices
- Dark/light mode support
- Project showcase with detailed descriptions
- Interactive experience timeline
- Contact form

## Local Development
```bash
# Install dependencies
npm install

# Start development server
npm start
```
Visit http://localhost:3000
## Update and Reployment
```
# 1. Add your changes
git add .

# 2. Commit your changes
git commit -m "Update portfolio styling and content"

# 3. Pull any remote changes first
git pull origin main

# 4. Push your changes
git push origin main

# 5. Deploy to GitHub Pages
npm run deploy
```
## Deployment
```bash
# Deploy to GitHub Pages
npm run deploy
```
